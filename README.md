# AI Automation Starter Kit for Coaches & Agencies

An open-source collection of automation workflows designed to help coaches, agencies, and consultants automate lead generation, follow-ups, webinar funnels, and customer engagement.

---

## 🚀 Features

- WhatsApp automation workflows
- Email follow-up sequences
- Webinar funnel automation
- AI chatbot templates
- Lead nurturing automation
- CRM workflow examples

---

## 📂 Project Structure

- whatsapp-workflows.md
- email-automation.md
- webinar-funnel-flow.md
- chatbot-prompts.md
- lead-followup-system.md
- sample-workflows.json
